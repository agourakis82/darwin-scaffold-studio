EXPORTED FUNCTIONS
------------------
- get_extended_tissue(id::String) -> OBOTerm or nothing
- list_extended_tissues(category::Symbol) -> Vector{OBOTerm}
  Categories: :all, :dental, :endocrine, :reproductive, :organ_substructures, :specialized_connective, :embryonic
- count_extended_tissues() -> Dict with counts per category

USAGE EXAMPLE
-------------
using .TissueLibraryExtended

# Get specific tissue
enamel = get_extended_tissue("UBERON:0001752")
println(enamel.name)  # "dental enamel"
println(enamel.definition)  # "Hypermineralized substance..."

# List all dental tissues
dental = list_extended_tissues(:dental)

# Get counts
counts = count_extended_tissues()
